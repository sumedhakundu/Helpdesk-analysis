# IT Helpdesk Ticket Analysis

## Problem Statement
IT teams lack visibility into ticket resolution performance across 100,000+ support tickets.

## Tools Used
- MySQL — data extraction & querying
- Python (pandas, matplotlib) — analysis & visualization
- Excel — interactive dashboard

## Key Findings
- "How-to" tickets are most common (12,739)
- Low priority tickets take longest to resolve (55 hrs)
- 50% of all tickets are successfully resolved
- Enterprise customers have slowest resolution time

## Recommendations
- Reassign staff from low to urgent priority tickets
- Set SLA alerts for tickets open more than 7 days

## Files
- queries.sql — all 5 SQL queries
- import_data.py — Python analysis script
- helpdesk_dashboard.xlsx — Excel dashboard
